# Chấm công

## Check in / out

- **Nhân sự › Chấm công › Check IN/OUT** (Kiosk hoặc app)
- Hoặc **Attendances** trên máy tính

## Quản lý xem báo cáo

**Attendances › Overview** — giờ vào/ra theo ngày.

## Overtime & Rules

**Cấu hình › Attendance** — grace time, overtime rules (tùy bản).

!!! note "Tích hợp máy chấm công"
    Cần module bridge hoặc import CSV tùy nhà cung cấp.
