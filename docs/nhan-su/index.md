| App | Chức năng |
|-----|-----------|
| **Employees** | Hồ sơ nhân viên |
| **Attendances** | Chấm công |
| **Time Off** | Nghỉ phép |
| **Recruitment** | Tuyển dụng |
| **Payroll** | Lương (localization) |
