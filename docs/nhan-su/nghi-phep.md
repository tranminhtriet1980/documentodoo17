# Nghỉ phép (Time Off)

## Tạo yêu cầu nghỉ

1. **Nhân sự › Nghỉ phép › Yêu cầu của tôi** → **Mới**
2. Chọn **Loại nghỉ** (Annual, Sick, Unpaid…)
3. **Start / End date**, mô tả
4. **Confirm** → chờ **Manager Approve**

## Phê duyệt (manager)

**Nghỉ phép › Cần phê duyệt** → **Approve** / **Refuse**

## Số dư phép

**Allocation** — HR cấp ngày phép đầu năm hoặc theo tháng.

## Lịch team

**Time Off › Calendar** — xem trùng lịch cả nhóm.
