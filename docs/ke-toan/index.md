# Kế toán

Module **Accounting / Invoicing** quản lý hóa đơn, thanh toán, sổ cái và báo cáo tài chính.

## Phân loại cài đặt

| Gói | Phạm vi |
|-----|---------|
| **Invoicing** | Hóa đơn khách/NCC cơ bản |
| **Accounting** | Sổ cái đầy đủ, báo cáo, đối soát |

## Thiết lập ban đầu (admin)

1. **Kế toán › Cấu hình › Cài đặt**
2. Chọn **Chart of Accounts** (VN template nếu có)
3. **Fiscal Localization** — thuế GTGT VN
4. Kỳ kế toán, lock date
