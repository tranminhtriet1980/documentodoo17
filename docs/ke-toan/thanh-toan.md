# Thanh toán

!!! tip "Quy trình Edupath"
    Chi tiết Payment, phiếu thu/chi, phê duyệt DNTT: [Quy trình nghiệp vụ kế toán](quy-trinh-nghiep-vu.md).

## Đăng ký thanh toán khách

1. Mở **Customer Invoice** trạng thái **Posted**
2. **Register Payment**
3. Chọn **Journal** (ngân hàng/tiền mặt), **Amount**, **Date**
4. **Create Payment** — invoice → **Paid**

## Thanh toán NCC

**Vendor Bill** → **Register Payment** — tương tự.

## Đối soát ngân hàng

**Kế toán › Ngân hàng › Sổ phụ**:

1. Import sao kê (OFX/CSV) hoặc đồng bộ
2. **Reconcile** với hóa đơn/chi phí

!!! tip "Batch payment"
    Thanh toán hàng loạt qua SEPA / export file (tùy localization).
