# Sổ cái (Journal Entries)

## Bút toán nhật ký

**Kế toán › Kế toán › Bút toán** → **Mới**

- **Journal** — loại (Bán hàng, Mua hàng, Ngân hàng…)
- **Lines** — Tài khoản Nợ/Có, đối tác, nhãn phân tích

## Tự động từ nghiệp vụ

Hóa đơn, thanh toán, kho (valuation) tạo bút toán khi **Post**.

## Analytic Accounting

Gắn **Analytic Account** / **Project** trên dòng để báo cáo theo trung tâm chi phí.

## Sửa / hủy

- **Reset to Draft** (nếu được phép)
- **Reverse Entry** — bút toán đảo

!!! warning "Audit"
    Mọi thay đổi lưu trong chatter và audit log.
