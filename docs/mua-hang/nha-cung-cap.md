# Nhà cung cấp

## Tạo nhà cung cấp

1. **Mua hàng › Nhà cung cấp › Nhà cung cấp** → **Mới**
2. Hoặc tạo từ **Contacts** với tick **Vendor**

## Thông tin quan trọng

| Trường | Ghi chú |
|--------|---------|
| **Tên / Công ty** | Tên pháp nhân |
| **Mã số thuế** | Xuất hóa đơn đầu vào |
| **Payment Terms** | Điều khoản thanh toán mặc định |
| **Purchase currency** | Ngoại tệ nếu mua quốc tế |
| **Bank** | Thanh toán chuyển khoản |

## Đánh giá & cảnh báo

- **Vendor Pricelist** — giá mua theo NCC
- Cảnh báo khi đặt hàng dưới MOQ (tùy cấu hình)
