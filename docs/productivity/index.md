# Năng suất & cộng tác

Các công cụ hỗ trợ làm việc nhóm không gắn trực tiếp một module nghiệp vụ.

| Công cụ | Mô tả |
|---------|--------|
| **Discuss** | Chat, kênh, tin nhắn trực tiếp |
| **Calendar** | Lịch, họp, nhắc lịch |
| **Documents** | Lưu trữ file (nếu cài) |
| **Knowledge** | Wiki nội bộ (nếu cài) |
