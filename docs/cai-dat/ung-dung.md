# Quản lý ứng dụng

## Cài ứng dụng mới

1. **Cài đặt › Ứng dụng**
2. Tìm module (CRM, Inventory…)
3. **Cài đặt** / **Upgrade**

## Gỡ / vô hiệu

Thường **không gỡ** app đang có dữ liệu — chỉ ẩn menu bằng quyền.

## Phụ thuộc

Odoo tự cài module phụ thuộc (ví dụ: Sales → cần Sales Teams).

## Custom module

Module riêng của công ty — cần dev cài qua Apps hoặc command line.
