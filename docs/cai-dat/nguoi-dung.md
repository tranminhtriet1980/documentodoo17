# Người dùng

## Tạo user

**Cài đặt › Người dùng & Công ty › Người dùng** → **Mới**

- **Name**, **Email** (login)
- **Access Rights** — tick nhóm quyền
- **Companies** — công ty được phép

## Mời qua email

**Send an invitation email** — user tự đặt mật khẩu.

## API keys

**Preferences › Account Security** — API key cho tích hợp (nếu bật).

## Vô hiệu hóa

**Archive user** — không đăng nhập, giữ lịch sử audit.
