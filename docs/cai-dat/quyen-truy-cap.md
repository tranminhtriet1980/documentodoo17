# Quyền truy cập

## Nhóm quyền (Groups)

**Cài đặt › Người dùng › Groups** — xem quyền từng nhóm.

Ví dụ CRM:

- **Sales / User: Own Documents Only**
- **Sales / User: All Documents**
- **Sales / Administrator**

## Record rules

Giới hạn dòng dữ liệu (chỉ lead của mình, chỉ công ty X…) — cấu hình trong Technical.

## Multi-company rules

User chỉ thấy dữ liệu công ty được gán.

## Portal & Public

- **Portal** — khách xem đơn, hóa đơn
- **Public** — website visitor

## Best practice

- Nguyên tắc **least privilege** — đủ quyền làm việc
- Tách admin nghiệp vụ và admin kỹ thuật
