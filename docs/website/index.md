# Website

Xây **website** doanh nghiệp và **cửa hàng trực tuyến** (eCommerce) trên Odoo.

## Bắt đầu

1. **Cài đặt › Website**
2. **Website › Site › Go to Website** — chế độ chỉnh sửa trực quan
3. Chọn **Theme**, trang **Home**, menu

## Mục lục

- [Trang & chỉnh sửa](trang-web.md)
- [Cửa hàng trực tuyến](cua-hang.md)

## Multi-website

Nhiều domain / thương hiệu — chọn website trên thanh editor.
