# Trang web & nội dung

## Chế độ chỉnh sửa

**Edit** trên website → kéo thả **Building Blocks**:

- Text, Image, Banner, Form
- **Dynamic snippets** — sản phẩm, blog, events

## Trang mới

**Website › Site › Pages** → **New** — slug URL, template.

## Blog & Forum

Cài **Blog**, **Forum** — quản lý bài viết, SEO meta.

## SEO

Tab **Promote** — meta title, description, URL redirect.

## Form liên hệ

Block **Form** → tạo **Lead** trong CRM khi submit.

!!! tip "Đa ngôn ngữ"
    **Translate** từng khối nội dung cho từng ngôn ngữ website.
