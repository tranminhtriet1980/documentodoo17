# Cửa hàng trực tuyến (eCommerce)

## Cài eCommerce

**Cài đặt › eCommerce** (hoặc bật trong Website settings).

## Sản phẩm online

Trên **Product** → tab **Sales** → **Published** on website.

## Giỏ hàng & thanh toán

- **Payment Providers** — chuyển khoản, PayPal, Stripe…
- **Delivery Methods** — phí ship theo vùng

## Đơn hàng web

Khách đặt → **Sales Order** trong backend → xử lý giao hàng như đơn thường.

## Khuyến mãi

**Coupons & Promotions** — mã giảm giá, chương trình loyalty (tùy module).
