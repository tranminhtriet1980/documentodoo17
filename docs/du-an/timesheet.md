# Timesheet

## Ghi giờ

1. **Dự án › Timesheets** → **Mới**
2. Chọn **Project**, **Task**, **Date**, **Hours**
3. Mô tả công việc

## Timer

Trên task: **Start** timer → **Stop** → tạo dòng timesheet.

## Phê duyệt

Manager **Validate** timesheet trước khi xuất hóa đơn.

## Xuất hóa đơn

**Sales** — invoice policy **Based on Timesheets** trên SO dịch vụ.

!!! tip "Grid view"
    Nhập nhanh cả tuần trên lưới timesheet.
