# Nhiệm vụ (Tasks)

## Tạo task

**Dự án › Dự án** → chọn project → **Mới** task

Hoặc **Dự án › Nhiệm vụ** — tất cả task.

## View

| View | Dùng khi |
|------|----------|
| **Kanban** | Theo stage (In Progress, Done…) |
| **List** | Lọc hàng loạt |
| **Gantt** | Timeline (Enterprise) |
| **Calendar** | Deadline |

## Sub-tasks & Dependencies

- **Sub-tasks** — chia nhỏ công việc
- **Blocked by** — phụ thuộc task khác

## Chatter & Files

Đính kèm tài liệu, @mention đồng nghiệp trong chatter.
