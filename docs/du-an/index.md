# Dự án (Project / QLHS)

Quản lý **hồ sơ** (XLHS — xử lý hồ sơ du học, du lịch, định cư), **nhiệm vụ**, **timesheet** và phối hợp nhóm.

## Cài đặt

**Cài đặt › Project** — có thể kèm **Timesheets**, tích hợp **CRM**, **Sales**, **Kế toán**.

## Workflow điển hình

1. **SO** được duyệt → khởi tạo **hồ sơ** (Project) theo template chương trình
2. **Tiếp nhận** — gán Admission, Leader…; khởi tạo **Task**
3. Thực hiện **Task** / stage — My Task, SharePoint, Activities
4. Chuyển **trạng thái** khi đủ điều kiện (ràng buộc)
