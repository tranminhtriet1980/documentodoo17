# Kiểm kê (Inventory Adjustment)

## Kiểm kê định kỳ

1. **Kho › Thao tác › Inventory Adjustments** (hoặc Physical Inventory)
2. Chọn **Location**, **Products**
3. Nhập **Counted Quantity**
4. **Apply** — tạo bút toán chênh lệch (nếu có accounting)

## Cycle Count

Chia vùng kiểm đếm luân phiên thay vì kiểm toàn bộ một lần.

!!! note "Quyền"
    Điều chỉnh tồn thường cần quyền **Inventory Manager**.

## Nguyên nhân lệch thường gặp

- Xuất/nhập chưa validate
- Hư hỏng chưa ghi nhận
- Nhầm location / lot
