# Nhập kho

## Nguồn nhập phổ biến

- **Purchase Receipt** — từ đơn mua
- **Manufacturing** — thành phẩm sản xuất
- **Inventory Adjustment** — điều chỉnh tăng
- **Returns** — khách trả hàng

## Thao tác Receipt

1. **Kho › Thao tác › Receipts**
2. Mở phiếu → kiểm tra **Demand** vs **Done**
3. **Validate** — hoàn tất nhập

## Lot / Serial

Nếu bật tracking: nhập **Lot Number** hoặc **Serial** trước Validate.

!!! warning "Không đủ tồn khi trả"
    Kiểm tra location trả hàng và policy return trên đơn bán.
