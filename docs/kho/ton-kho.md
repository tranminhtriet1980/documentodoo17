# Tồn kho

## Xem tồn

- **Kho › Báo cáo › Stock On Hand**
- Trên **Product** → smart button **On Hand**

## Dự trữ (Reserved)

- **Forecasted** = On Hand − Reserved + Incoming
- Đơn bán confirm → reserved cho delivery

## Reordering Rules

**Kho › Cấu hình › Reordering Rules**:

- **Min / Max quantity**
- Tự tạo **Purchase** hoặc **Manufacturing** khi dưới min (cần scheduler)

## Valuation

Giá trị tồn (FIFO, AVCO) — xem tab **Inventory Valuation** nếu bật accounting anglosaxon.
