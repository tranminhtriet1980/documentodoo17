# Tính năng (Functional)

Phần **Functional** mô tả **cách sử dụng tính năng** Odoo 17 trong công việc hàng ngày: quy trình nghiệp vụ, màn hình, nút bấm, báo cáo.

## Ai nên đọc phần này?

- Nhân viên **Bán hàng / CRM**
- **Kế toán**, **thủ kho**, **mua hàng**
- **HR**, **marketing**, **dự án**
- **Quản trị viên nghiệp vụ** (cài app, tạo user, phân quyền cơ bản)

## Không nằm trong phần Functional

Các chủ đề sau thuộc tab **[Kỹ thuật (Technical)](../technical/index.md)**:

- Cài Odoo trên server, file `odoo.conf`
- Backup PostgreSQL, restore
- Viết module Python, kế thừa view XML
- XML-RPC / API tích hợp hệ thống ngoài
