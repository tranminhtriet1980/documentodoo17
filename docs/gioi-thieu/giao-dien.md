# Giao diện Odoo 17

## Thanh điều hướng chính

| Vùng | Chức năng |
|------|-----------|
| **Menu ứng dụng** (góc trên trái, lưới ô) | Chuyển giữa CRM, Bán hàng, Kho, Kế toán… |
| **Thanh menu** | Menu của ứng dụng hiện tại (Bán hàng, Cấu hình…) |
| **Thanh tìm kiếm** | Tìm bản ghi, lệnh, liên hệ… |
| **Biểu tượng tin nhắn / hoạt động** | Thảo luận, hoạt động gần đây |
| **Avatar người dùng** | Tùy chọn tài khoản, đăng xuất, chuyển công ty |

## Các dạng xem (Views)

### List (Danh sách)

- Xem nhiều bản ghi dạng bảng
- Lọc, nhóm theo cột, xuất Excel (nếu được phép)
- Click dòng để mở **form**

### Form (Biểu mẫu)

- Chi tiết một bản ghi
- **Chatter** (bên phải hoặc dưới): ghi chú, gửi email, đính kèm file, theo dõi hoạt động

### Kanban

- Thẻ kéo thả giữa các cột (giai đoạn)
- Dùng nhiều trong CRM pipeline

### Calendar / Graph / Pivot

- Lịch, biểu đồ, bảng tổng hợp — tùy module

## Thao tác thường dùng

| Thao tác | Cách làm |
|----------|----------|
| **Tạo mới** | Nút **Mới** / **Create** |
| **Lưu** | Tự lưu khi rời trường hoặc nút **Lưu** (tùy màn hình) |
| **Hủy thay đổi** | **Discard** hoặc đóng không lưu |
| **In / PDF** | Nút **In** trên form hoặc list |
| **Hành động** | Menu **Action** trên list (xuất, xóa hàng loạt…) |

## Breadcrumb & URL

Odoo lưu trạng thái trên URL. Bạn có thể **bookmark** một màn hình cụ thể (một pipeline, một báo cáo) để mở lại nhanh.

!!! tip "Phím tắt"
    - `Alt + B` — mở breadcrumb (tùy cấu hình trình duyệt)
    - Trong list: chọn nhiều dòng bằng checkbox đầu bảng

## Chế độ sáng / tối

Vào **Preferences / Tùy chọn** từ menu avatar (nếu admin bật) để đổi theme giao diện Odoo.
